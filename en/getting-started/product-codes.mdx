AllTick uses a unified code format to identify financial products. Codes are case-sensitive and must match the formats below exactly.

## Code formats

| Market | Format | Examples |
|--------|--------|----------|
| US stocks | `{TICKER}.US` | AAPL.US, TSLA.US, GOOGL.US |
| HK stocks | `{CODE}.HK` | 700.HK, 9988.HK, 1299.HK |
| A-shares (SSE) | `{CODE}.SH` | 600519.SH, 601318.SH |
| A-shares (SZSE) | `{CODE}.SZ` | 000858.SZ, 000001.SZ |
| Forex | `{BASE}{QUOTE}` | EURUSD, USDJPY, GBPUSD |
| Crypto | `{BASE}{QUOTE}` | BTCUSDT, ETHUSDT, SOLUSDT |
| Precious metals | Fixed codes | GOLD, SILVER |
| Energy | Fixed codes | USOIL, UKOIL |
