## Step 1: Understand API Endpoints

Learn the URL structure and expected parameters. Read the [API Address Guide](/en/getting-started/api-address) to understand access URLs and required parameters.

## Step 2: Apply for Your Token

Obtain the credentials needed to access the API. Follow the [Token Application](/en/getting-started/token) guide to get your token.

## Step 3: Understand Rate Limits

Avoid request failures due to limit violations. Read the [Rate Limits](/en/getting-started/rate-limits) documentation.

## Step 4: Learn Request/Response Format

Ensure you can correctly construct requests and parse responses. See the [Standard Headers](/en/getting-started/standard-headers) documentation.

## Step 5: Choose Product Codes

Determine the specific financial products you want to query. Refer to the [Product Code List](/en/getting-started/product-codes).

## Step 6: Execute Requests

Combine the above steps to prepare your request URL and parameters, then send requests to get market data.

## Best Practices

- **Security**: Protect your Token from unauthorized access
- **Monitoring**: Regularly check for API documentation updates
- **Testing**: Validate requests in a test environment before production use
- **Support**: Contact technical support promptly when encountering issues
