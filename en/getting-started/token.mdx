AllTick authenticates all API requests using a token. Every request must include a valid token as a URL query parameter.

## Get your token

<Steps>
  <Step title="Register">
    Go to [alltick.co/register](https://alltick.co/register) and sign up with your email.
  </Step>
  <Step title="Open dashboard">
    After registration, you are automatically redirected to the dashboard.
  </Step>
  <Step title="Copy token">
    Find your token in the **API Keys** section and copy it.
  </Step>
</Steps>

## Usage

**REST API** — pass as the `token` query parameter:

```
https://quote.alltick.co/quote-b-api/kline?token=YOUR_TOKEN&query=...
```

**WebSocket API** — pass as a query parameter in the connection URL:

```
wss://quote.alltick.co/quote-b-ws-api?token=YOUR_TOKEN
```

## Security best practices

- Never expose your token in client-side code or public repositories
