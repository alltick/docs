---
title: 'Trade Subscription'
description: 'Subscribe to real-time trade tick data'
---

Subscribe to latest trade prices (real-time tick data). Each subscription request overwrites the previous one.

<Warning>
To add symbols, resend the full list. For example, if subscribed to A, B, C and you want to add D, send A, B, C, D.
</Warning>

## Request (cmd_id: 22004)

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| cmd_id | integer | Yes | Fixed: 22004 |
| seq_id | integer | Yes | Sequence number, returned in response |
| trace | string | Yes | Correlation ID, must be unique |
| data.symbol_list | array | Yes | Symbol list |
| data.symbol_list[].code | string | Yes | Product code |

```json
{
  "cmd_id": 22004,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {
    "symbol_list": [
      { "code": "BTCUSDT" },
      { "code": "ETHUSDT" }
    ]
  }
}
```

## Response (cmd_id: 22005)

```json
{
  "ret": 200,
  "msg": "ok",
  "cmd_id": 22005,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```

## Push (cmd_id: 22998)

| Field | Type | Description |
|-------|------|-------------|
| code | string | Product code |
| seq | string | Quote sequence number |
| tick_time | string | Timestamp (milliseconds) |
| price | string | Latest trade price |
| volume | string | Trade volume |
| turnover | string | Trade turnover |
| trade_direction | integer | 0=neutral, 1=active buy, 2=active sell |

```json
{
  "cmd_id": 22998,
  "data": {
    "code": "BTCUSDT",
    "seq": "1605509068000001",
    "tick_time": "1605509068",
    "price": "651.12",
    "volume": "300",
    "turnover": "12345.6",
    "trade_direction": 1
  }
}
```
