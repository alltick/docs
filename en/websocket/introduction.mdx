AllTick WebSocket API provides real-time market data push services. Subscribe to symbols after connecting to continuously receive latest trade prices and order book depth.

## Protocol commands

| Command ID | Description |
|------------|-------------|
| 22000/22001 | Heartbeat (Ping/Pong) |
| 22002/22003 | Depth subscription |
| 22004/22005 | Trade subscription |
| 22006/22007 | Cancel subscription |
| 22998 | Trade price push |
| 22999 | Order book push |

## Endpoints

| Asset class | Connection URL |
|-------------|----------------|
| US stocks, HK stocks, A-shares, indices | `wss://quote.alltick.co/quote-stock-b-ws-api?token=YOUR_TOKEN` |
| Forex, crypto, precious metals, commodities | `wss://quote.alltick.co/quote-b-ws-api?token=YOUR_TOKEN` |

## Usage guide

1. Include your token as a URL query parameter when connecting
2. Send a heartbeat every **10 seconds** after connection
3. Connection is closed if no heartbeat received within **30 seconds**
4. Each subscription request overwrites the previous one (resend the full list to add symbols)
5. Implement automatic reconnection logic in your client
