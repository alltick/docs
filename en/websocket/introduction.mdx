## Overview

AllTick WebSocket API provides real-time market data push services:

| Command ID | Description |
|------------|-------------|
| 22000/22001 | Heartbeat |
| 22002/22003 | Order book subscription |
| 22004/22005 | Trade price subscription |
| 22006/22007 | Cancel subscription |
| 22998 | Trade price push |
| 22999 | Order book push |

## Endpoints

**Stocks (US, HK, A-shares):**

```
wss://quote.alltick.co/quote-stock-b-ws-api?token=YOUR_TOKEN
```

**Forex, Crypto, Precious Metals, Commodities:**

```
wss://quote.alltick.co/quote-b-ws-api?token=YOUR_TOKEN
```

## Usage Guide

1. Include your authentication token in the URL when establishing the WebSocket connection
2. Send a heartbeat every **10 seconds** after connection
3. Connection will be closed if no heartbeat received within **30 seconds**
4. Each subscription request overwrites the previous one (to add products, resend the full list)
5. Implement automatic reconnection logic in your client
