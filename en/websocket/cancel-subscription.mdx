---
title: 'Cancel Subscription'
description: 'Unsubscribe from market data push'
---

Cancel active quote subscriptions.

## Request (cmd_id: 22006)

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| cmd_id | integer | Yes | Fixed: 22006 |
| seq_id | integer | Yes | Sequence number |
| trace | string | Yes | Correlation ID |
| data.cancel_type | integer | Yes | 0=all, 1=depth, 2=trade, 3=forex |

```json
{
  "cmd_id": 22006,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {
    "cancel_type": 1
  }
}
```

## Response (cmd_id: 22007)

```json
{
  "ret": 200,
  "msg": "ok",
  "cmd_id": 22007,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```
