---
title: 'Heartbeat'
description: 'Keep-alive ping/pong mechanism'
---

Send a heartbeat every 10 seconds. The server closes the connection after 30 seconds without a heartbeat.

## Request (cmd_id: 22000)

```json
{
  "cmd_id": 22000,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```

## Response (cmd_id: 22001)

```json
{
  "ret": 200,
  "msg": "ok",
  "cmd_id": 22001,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```
