## Welcome

Welcome to the AllTick API documentation! Here you'll find tutorials, overviews, feature introductions, and use cases.

## What is AllTick

[AllTick](https://alltick.co/) provides a complete financial market tick data solution, covering forex, HK stock CFDs, US stock CFDs, commodities, and cryptocurrency market data APIs. Designed for exchanges, developers, quant teams, fintech companies, and professional institutions.

## Key Features

- **Comprehensive Coverage**: Real-time and historical data for forex, stocks, commodities, and cryptocurrencies
- **Multi-language SDKs**: Client libraries in multiple programming languages for quick integration
- **WebSocket & REST API**: Low-latency, high-reliability data delivery via both protocols
- **High Reliability**: Stable service with accurate and complete data
- **Custom Solutions**: Tailored services for professional institutions and fintech companies
