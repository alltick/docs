AllTick is a financial market data platform that delivers tick-level quotes, K-line history, order book depth, and trade prices across global asset classes through REST and WebSocket APIs.

Whether you are building a trading dashboard, running quantitative research, or powering a fintech product, AllTick gives you a single integration point for real-time and historical data from equities, forex, crypto, and more.

## Core capabilities

| Capability | Description |
|------------|-------------|
| Real-time streaming | WebSocket push with ~170ms average latency |
| Historical K-lines | Minute, hour, day, week, and month granularities |
| Order book depth | Multi-level bid/ask data (depth varies by market) |
| Latest trade price | REST endpoint for point-in-time tick lookups |
| Multi-language examples | Complete code samples in Go, Python, Java, and PHP |

## Supported markets

| Market | Description | Example symbols |
|--------|-------------|-----------------|
| Equities | US stocks (CFD), HK stocks (CFD), A-shares | AAPL.US, 700.HK, 600519.SH |
| Forex | 100+ major and cross currency pairs | EURUSD, USDJPY, GBPUSD |
| Crypto | Major CEX trading pairs | BTCUSDT, ETHUSDT, SOLUSDT |
| Precious metals & commodities | Gold, silver, oil | GOLD, SILVER, USOIL |
| Global indices | Major stock indices | Dow Jones, S&P 500, Hang Seng |

## API access methods

AllTick exposes two complementary protocols:

| Protocol | Best for | Characteristics |
|----------|----------|-----------------|
| REST API | Historical K-line queries, one-time price lookups, server-side pipelines | Pull on demand, simple and direct |
| WebSocket API | Live dashboards, price alerts, continuous data streams | Subscribe and receive, no polling |
