## Application Process

### Step 1 - Fill in registration information

<Note>
Token application URL: <a href="https://alltick.co/register">https://alltick.co/register</a>
</Note>

1. **Visit the registration page**: Go to the [Token application URL](https://alltick.co/register) to start registration.
2. **Enter registration details**:
   * **Email address**: Enter a valid email address for future communication and account verification.
   * **Password**: Create a secure password. Use uppercase and lowercase letters, special characters, and numbers to improve account security.
   * **Email verification code**: Enter the verification code sent to your email. If you do not see the code, check your spam folder.
3. **Complete registration**: After confirming the information is correct, click the registration button. The system will process the registration and redirect you to the dashboard after success.

<img src="https://apis.alltick.co/files/HU2q7WCSdCJbSIRnXGZG" alt="" />

## Step 2 - Get the API Token

1. **Open the dashboard**: After registration, the system automatically navigates to the dashboard.
2. **Find the Token**: In the dashboard, go to the API keys section. You will find the newly generated API Token there.
3. **Record the Token**: Copy and store your Token securely. You need this Token for authentication when calling APIs.

<img src="https://apis.alltick.co/files/kZJy1HkkEbyo16ROONXf" alt="" />

#### Notes

* Keep your Token secure and do not share it with untrusted third parties.
* If you encounter any issue during registration or Token generation, contact customer support immediately.
* Always use the latest Token for API calls to avoid data access issues caused by expiration or invalidation.

By following the steps above, you can complete the Token application process and obtain the permissions required to access the API.
