AllTick は、REST と WebSocket API を通じて、ティックレベルの相場、Kライン履歴、板情報、最新約定価格をグローバルな資産クラスにわたって提供する金融市場データプラットフォームです。

トレーディングダッシュボードの構築、クオンツリサーチの実行、フィンテック製品へのデータ提供など、AllTick は株式、外国為替、暗号通貨などのリアルタイム・履歴データを単一の統合ポイントで提供します。

## コア機能

| 機能 | 説明 |
|------|------|
| リアルタイムストリーミング | WebSocket プッシュ、平均遅延約 170ms |
| 履歴 K ライン | 分足、時間足、日足、週足、月足に対応 |
| 板情報（Order Book） | マルチレベルの売買板データ（深度は市場により異なる） |
| 最新約定価格 | REST エンドポイントでポイントインタイム Tick 照会 |
| 多言語サンプル | Go、Python、Java、PHP の完全なコード例 |

## 対応市場

| 市場 | 説明 | 代表銘柄 |
|------|------|----------|
| 株式 | 米国株 CFD、香港株 CFD、A 株 | AAPL.US, 700.HK, 600519.SH |

| 暗号通貨 | 主要 CEX 取引ペア | BTCUSDT, ETHUSDT, SOLUSDT |

| グローバル指数 | 主要株価指数 | ダウ平均, S&P 500, ハンセン指数 |

## アクセス方法

AllTick は 2 つの補完的なプロトコルを提供します：

| プロトコル | 最適な用途 | 特徴 |
|-----------|-----------|------|
| REST API | 履歴 K ライン照会、一回限りの価格照会、サーバーサイドパイプライン | オンデマンドプル、シンプル |
| WebSocket API | ライブダッシュボード、価格アラート、継続的データストリーム | サブスクライブ＆受信、ポーリング不要 |
