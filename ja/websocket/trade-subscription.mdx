---
title: '約定価格購読'
description: 'リアルタイム Tick データの購読'
---

製品の最新約定価格（リアルタイム Tick）を購読します。各購読リクエストは前回を上書きします。

<Warning>
銘柄を追加する場合は完全なリストを再送信してください。例：A、B、C を購読中に D を追加する場合、A、B、C、D を送信。
</Warning>

## リクエスト（cmd_id: 22004）

| フィールド | 型 | 必須 | 説明 |
|-----------|------|------|------|
| cmd_id | integer | はい | 固定値 22004 |
| seq_id | integer | はい | シーケンス番号 |
| trace | string | はい | 追跡 ID（一意） |
| data.symbol_list | array | はい | 銘柄リスト |
| data.symbol_list[].code | string | はい | 製品コード |

```json
{
  "cmd_id": 22004,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {
    "symbol_list": [
      { "code": "BTCUSDT" },
      { "code": "ETHUSDT" }
    ]
  }
}
```

## レスポンス（cmd_id: 22005）

```json
{
  "ret": 200,
  "msg": "ok",
  "cmd_id": 22005,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```

## プッシュ（cmd_id: 22998）

| フィールド | 型 | 説明 |
|-----------|------|------|
| code | string | 製品コード |
| seq | string | 相場シーケンス番号 |
| tick_time | string | タイムスタンプ（ミリ秒） |
| price | string | 最新約定価格 |
| volume | string | 出来高 |
| turnover | string | 売買代金 |
| trade_direction | integer | 0=中立、1=能動的買い、2=能動的売り |

```json
{
  "cmd_id": 22998,
  "data": {
    "code": "BTCUSDT",
    "seq": "1605509068000001",
    "tick_time": "1605509068",
    "price": "651.12",
    "volume": "300",
    "turnover": "12345.6",
    "trade_direction": 1
  }
}
```
