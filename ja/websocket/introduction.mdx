title: 'WebSocket API 概要'

---

AllTick WebSocket API はリアルタイムマーケットデータプッシュサービスを提供します。接続後に銘柄を購読すると、最新約定価格と板情報を継続的に受信できます。

## プロトコルコマンド

| コマンド ID | 説明 |
|------------|------|
| 22000/22001 | ハートビート（Ping/Pong） |
| 22002/22003 | 板情報購読 |
| 22004/22005 | 約定価格購読 |
| 22006/22007 | 購読キャンセル |
| 22998 | 約定価格プッシュ |
| 22999 | 板情報プッシュ |

## エンドポイント

| 資産クラス | 接続 URL |
|-----------|----------|
| 米国株、香港株、A 株、指数 | `wss://quote.alltick.co/quote-stock-b-ws-api?token=YOUR_TOKEN` |
| 外国為替、暗号通貨、貴金属、コモディティ | `wss://quote.alltick.co/quote-b-ws-api?token=YOUR_TOKEN` |

## 使用ガイド

1. 接続時に URL クエリパラメータとして Token を含める
2. 接続後、**10 秒**ごとにハートビートを送信
3. **30 秒**以内にハートビートがないと接続切断
4. 各購読リクエストは前回を上書き（銘柄追加時は完全なリストを再送信）
5. クライアントに自動再接続ロジックを実装すること
