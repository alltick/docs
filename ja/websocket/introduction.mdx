## 概要

AllTick WebSocket API はリアルタイムマーケットデータプッシュサービスを提供します：

| コマンドID | 説明 |
|-----------|------|
| 22000/22001 | ハートビート |
| 22002/22003 | オーダーブック購読 |
| 22004/22005 | 約定価格購読 |
| 22006/22007 | 購読キャンセル |
| 22998 | 約定価格プッシュ |
| 22999 | オーダーブックプッシュ |

## エンドポイント

**株式（米国株、香港株、A株）：**

```
wss://quote.alltick.co/quote-stock-b-ws-api?token=YOUR_TOKEN
```

**外国為替、暗号通貨、貴金属、コモディティ：**

```
wss://quote.alltick.co/quote-b-ws-api?token=YOUR_TOKEN
```

## 使用ガイド

1. WebSocket接続確立時にURLに認証Tokenを含める
2. 接続後、**10秒**ごとにハートビートを送信
3. **30秒**以内にハートビートが受信されない場合、接続が切断される
4. 各購読リクエストは前回の購読を上書き（製品を追加する場合は完全なリストを再送信）
5. クライアントに自動再接続ロジックを実装することを推奨
