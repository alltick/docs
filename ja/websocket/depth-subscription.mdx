---
title: '板情報購読'
description: 'リアルタイム板情報（Order Book）の購読'
---

製品の最新板情報（Order Book）を購読します。各購読リクエストは前回を上書きします。

市場別の最大板深度：

| 市場 | 最大深度 |
|------|----------|
| 外国為替、貴金属、原油 | 1 レベル |
| 暗号通貨 | 5 レベル |
| 香港株 | 10 レベル |
| 米国株 | 1 レベル |
| A 株 | 5 レベル |

## リクエスト（cmd_id: 22002）

| フィールド | 型 | 必須 | 説明 |
|-----------|------|------|------|
| cmd_id | integer | はい | 固定値 22002 |
| seq_id | integer | はい | シーケンス番号 |
| trace | string | はい | 追跡 ID |
| data.symbol_list | array | はい | 銘柄リスト |
| data.symbol_list[].code | string | はい | 製品コード |
| data.symbol_list[].depth_level | integer | いいえ | 深度レベル（デフォルト: 1） |

```json
{
  "cmd_id": 22002,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {
    "symbol_list": [
      { "code": "BTCUSDT", "depth_level": 5 },
      { "code": "ETHUSDT", "depth_level": 5 }
    ]
  }
}
```

## レスポンス（cmd_id: 22003）

```json
{
  "ret": 200,
  "msg": "ok",
  "cmd_id": 22003,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```

## プッシュ（cmd_id: 22999）

| フィールド | 型 | 説明 |
|-----------|------|------|
| code | string | 製品コード |
| seq | string | 相場シーケンス番号 |
| tick_time | string | タイムスタンプ（ミリ秒） |
| bids | array | 買い板 [{price, volume}] |
| asks | array | 売り板 [{price, volume}] |

```json
{
  "cmd_id": 22999,
  "data": {
    "code": "BTCUSDT",
    "seq": "1605509068000001",
    "tick_time": "1605509068",
    "bids": [
      { "price": "9.12", "volume": "100" }
    ],
    "asks": [
      { "price": "9.13", "volume": "150" }
    ]
  }
}
```
