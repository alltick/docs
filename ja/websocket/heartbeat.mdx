---
title: 'ハートビート'
description: 'WebSocket 接続維持のための Ping/Pong'
---

10 秒ごとにハートビートを送信します。30 秒間ハートビートがないとサーバーが接続を切断します。

## リクエスト（cmd_id: 22000）

```json
{
  "cmd_id": 22000,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```

## レスポンス（cmd_id: 22001）

```json
{
  "ret": 200,
  "msg": "ok",
  "cmd_id": 22001,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```
