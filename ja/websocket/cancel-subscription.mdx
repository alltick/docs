---
title: '購読キャンセル'
description: 'マーケットデータプッシュの購読解除'
---

アクティブな相場購読をキャンセルします。

## リクエスト（cmd_id: 22006）

| フィールド | 型 | 必須 | 説明 |
|-----------|------|------|------|
| cmd_id | integer | はい | 固定値 22006 |
| seq_id | integer | はい | シーケンス番号 |
| trace | string | はい | 追跡 ID |
| data.cancel_type | integer | はい | 0=全部、1=板情報、2=約定、3=為替 |

```json
{
  "cmd_id": 22006,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {
    "cancel_type": 1
  }
}
```
