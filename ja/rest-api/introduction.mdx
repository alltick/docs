## 概要

AllTick REST API ドキュメントには、HTTP リクエスト例と HTTP インターフェース API が含まれます。HTTP インターフェース API ページは公式 API ドキュメントに従って構成されており、Kライン、複数商品のKライン、最新板情報、最新約定価格、株式商品の基本情報、売買停止・再開情報照会、および騰落率、休場、休日などの説明を含みます。
