## GET /static_info

## インターフェース説明

このインターフェースは、米国株、香港株、A株商品の一部基本情報の一括照会のみ対応しています。

## **インターフェースURL**

* **ベースパス**: `/quote-stock-b-api/static_info`
* **完全URL**: `https://quote.alltick.co/quote-stock-b-api/static_info`

## **リクエスト例**

照会リクエストを送信する際は、メソッド名とtoken情報を含める必要があります。例:

```text
https://quote.alltick.co/quote-stock-b-api/static_info?token=your_token&query=queryData
```

## リクエストパラメータ

| 名称 | 位置 | 型 | 必須 | 説明 |
| --- | --- | --- | --- | --- |
| token | query | string | いいえ |  |
| query | query | string | いいえ | queryリクエストパラメータ説明を参照してください。 |

**次のJSONをURLエンコードし、URLのクエリ文字列パラメータ `query` に設定します。**

```json
{
  "trace": "edd5df80-df7f-4acf-8f67-68fd2f096426",
  "data": {
    "symbol_list": [
      {
        "code": "857.HK"
      },
      {
        "code": "UNH.US"
      }
    ]
  }
}
```
