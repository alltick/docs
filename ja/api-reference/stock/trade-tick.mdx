## GET /trade-tick

## インターフェース説明

このインターフェースは商品の最新約定価格（最新Tickデータ、現在値または最新価格）の一括照会に対応しています。履歴約定価格（履歴Tickデータ）の照会には対応していません。

## リクエスト例

**1. 米国株、香港株、A株、主要指数データ:**

照会リクエストを送信する際は、メソッド名とtoken情報を含めます。例:

[https://quote.alltick.co/quote-stock-b-api/trade-tick?token=your_token&query=queryData](https://quote.alltick.co/quote-stock-b-api/trade-tick?token=your_token&query=queryData)

**2. 外国為替、貴金属、暗号資産、原油、CFD指数、商品:**

照会リクエストを送信する際は、メソッド名とtoken情報を含めます。例:

[https://quote.alltick.co/quote-b-api/trade-tick?token=your_token&query=queryData](https://quote.alltick.co/quote-b-api/trade-tick?token=your_token&query=queryData)

## queryリクエストパラメータ

次のJSONをURLエンコードし、URLの `query` クエリ文字列に設定します。

```json
{
  "trace": "edd5df80-df7f-4acf-8f67-68fd2f096426",
  "data": {
    "symbol_list": [
      {
        "code": "857.HK"
      },
      {
        "code": "UNH.US"
      }
    ]
  }
}
```

| 名称 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| trace | string | はい |  |
| data | object | はい |  |
| » symbol_list | [object] | はい |  |
| »» code | string | いいえ | コード。<a href="https://docs.google.com/spreadsheets/d/1avkeR1heZSj6gXIkDeBt8X3nv4EzJetw4yFuKjSDYtA/edit?gid=495387863#gid=495387863">codeリスト</a>から照会するcodeを選択してください。<br />**注意: code値は大文字・小文字を区別し、商品リスト内のcodeと一致させる必要があります。** |

## レスポンスデータ構造

| 名称 | 型 | 必須 | 説明 |
| --- | --- | --- | --- |
| » ret | integer | true | 戻りコード |
| » msg | string | true | 戻りコードに対応するメッセージ |
| » trace | string | true | リクエストトレース |
| » data | object | true |  |
| »» tick_list | [object] | true |  |
| »»» code | string | false | コード |
| »»» seq | string | false | シーケンス番号 |
| »»» tick_time | string | false | タイムスタンプ |
| »»» price | string | false | 約定価格 |
| »»» volume | string | false | 約定数量 |
| »»» turnover | string | false | 約定金額 |
| »»» trade_direction | integer | false | 取引方向:<br />1. 0はデフォルト値、1は買い、2は売りです。<br />2. 外国為替、貴金属、エネルギーはデフォルトで1のみ返します。<br />3. 株式と暗号資産は市場状況により0、1、2を返す場合があります。<br />4. 詳細:<br />0: 中立。最良買気配と最良売気配の間の価格で約定したことを示します。<br />1: アグレッシブ買い。売気配価格またはそれ以上の価格で約定したことを示します。<br />2: アグレッシブ売り。買気配価格またはそれ以下の価格で約定したことを示します。 |
