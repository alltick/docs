## 申請フロー

### ステップ 1 - 登録情報を入力する

<Note>
Token 申請 URL：<a href="https://alltick.co/register">https://alltick.co/register</a>
</Note>

1. **登録ページへアクセス**：[Token 申請 URL](https://alltick.co/register) にアクセスして登録を開始します。
2. **登録情報を入力**：
   * **メールアドレス**：有効なメールアドレスを入力します。このアドレスは以後の連絡やアカウント認証に使用されます。
   * **パスワード設定**：安全なパスワードを作成します。大文字、小文字、特殊文字、数字を含めるとアカウントの安全性が高まります。
   * **メール認証コード**：メールに送信された認証コードを入力します。コードが見つからない場合は迷惑メールフォルダを確認してください。
3. **登録完了**：情報に誤りがないことを確認し、登録ボタンをクリックします。登録成功後、システムは自動的にダッシュボードへ移動します。

<img src="https://apis.alltick.co/files/HU2q7WCSdCJbSIRnXGZG" alt="" />

## ステップ 2 - API Token を取得する

1. **ダッシュボードへ入る**：登録完了後、システムは自動的にダッシュボードへ移動します。
2. **Token を探す**：ダッシュボード画面で API keys セクションへ移動します。ここで生成された API Token を確認できます。
3. **Token を記録する**：Token をコピーし、安全に保存してください。API 呼び出し時の認証にこの Token を使用します。

<img src="https://apis.alltick.co/files/kZJy1HkkEbyo16ROONXf" alt="" />

#### 注意事項

* Token を安全に保護し、信頼できない第三者と共有しないでください。
* 登録または Token 生成中に問題が発生した場合は、すぐにカスタマーサポートへお問い合わせください。
* 期限切れや無効化によるデータアクセスへの影響を避けるため、API 呼び出しには常に最新の Token を使用してください。

上記の手順に従うことで Token 申請を完了し、API へアクセスするために必要な権限を取得できます。
