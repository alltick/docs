## 株式市場データWebSocket購読

* **ベースパス**: `/quote-stock-b-ws-api`
* **完全URL**: `wss://quote.alltick.co/quote-stock-b-ws-api`

**認証情報**

接続を確立するたびに、次のようにURLへ認証tokenを付加する必要があります。

* **ベースパス**: `/quote-b-ws-api`
* **完全URL**: `wss://quote.alltick.co/quote-b-ws-api`

接続を確立する際も、データ転送の安全性を確保するため、URLへ認証tokenを付加する必要があります。正しい形式は次のとおりです。

