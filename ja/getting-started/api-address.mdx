---
title: 'API アドレス'
description: '資産クラス別の REST・WebSocket エンドポイント URL'
---

AllTick は資産クラスごとにエンドポイントを分けています。株式は専用の Base URL を使用し、その他の市場は共通の Base URL を共有します。

## REST API エンドポイント

| 資産クラス | Base URL |
|-----------|----------|
| 米国株、香港株、A 株、指数 | `https://quote.alltick.co/quote-stock-b-api` |
| 外国為替、暗号通貨、貴金属、コモディティ | `https://quote.alltick.co/quote-b-api` |

利用可能なパス：

| パス | メソッド | 説明 |
|------|---------|------|
| `/kline` | GET | 単一製品の履歴 K ライン照会 |
| `/batch-kline` | POST | 複数製品の最新 2 本 K ライン一括照会 |
| `/trade-tick` | GET | 最新約定価格照会 |
| `/depth-tick` | GET | 最新板情報照会 |

リクエスト例：

```
https://quote.alltick.co/quote-stock-b-api/kline?token=YOUR_TOKEN&query=URL_ENCODED_JSON
```

## WebSocket エンドポイント

| 資産クラス | 接続 URL |
|-----------|----------|
| 米国株、香港株、A 株、指数 | `wss://quote.alltick.co/quote-stock-b-ws-api?token=YOUR_TOKEN` |
| 外国為替、暗号通貨、貴金属、コモディティ | `wss://quote.alltick.co/quote-b-ws-api?token=YOUR_TOKEN` |

<Note>
外国為替や暗号通貨の銘柄を照会する場合は `/quote-b-api`（REST）または `/quote-b-ws-api`（WebSocket）を使用します。株式の場合は `-stock-` を含むパスを使用します。リクエストパラメータの構造は同一で、Base URL のみが異なります。
</Note>
