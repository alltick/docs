---
title: '標準ヘッダー'
description: 'HTTP および WebSocket の共通リクエスト/レスポンス形式'
---

## HTTP リクエスト標準ヘッダー

| フィールド | 型 | 必須 | 説明 |
|-----------|------|------|------|
| trace | string | はい | リクエスト者が生成する一意のID（最大64文字） |
| data | object | はい | データ本体（各インターフェースで定義） |

リクエスト例：

```json
{
  "trace": "c2a8a146-a647-4d6f-ac07-8c4805bf0b74",
  "data": {}
}
```

## HTTP レスポンス標準ヘッダー

| フィールド | 型 | 説明 |
|-----------|------|------|
| ret | int32 | エラーコード |
| msg | string | 成功/失敗の説明 |
| trace | string | リクエストと同じtrace |
| data | object | レスポンスデータ本体 |

レスポンス例：

```json
{
  "ret": 200,
  "msg": "ok",
  "trace": "c2a8a146-a647-4d6f-ac07-8c4805bf0b74",
  "data": {}
}
```

## WebSocket リクエスト形式

| フィールド | 型 | 必須 | 説明 |
|-----------|------|------|------|
| cmd_id | integer | はい | プロトコルコマンドID |
| seq_id | integer | はい | リクエストシーケンスID |
| trace | string | はい | ログ追跡用ID |
| data | object | はい | データ本体 |

## エラーコード

| コード | 説明 |
|--------|------|
| 200 | 成功 |
| 202 | リクエストパラメータ無効 |
