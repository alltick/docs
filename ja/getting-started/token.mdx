AllTick はすべての API リクエストを Token で認証します。各リクエストの URL クエリパラメータに有効な Token を含める必要があります。

## Token の取得

<Steps>
  <Step title="アカウント登録">
    [alltick.co/register](https://alltick.co/register) にアクセスし、メールアドレスで登録します。
  </Step>
  <Step title="ダッシュボードを開く">
    登録完了後、自動的にダッシュボードにリダイレクトされます。
  </Step>
  <Step title="Token をコピー">
    **API Keys** セクションで Token を見つけてコピーします。
  </Step>
</Steps>

## 使用方法

**REST API** — `token` クエリパラメータとして渡す：

```
https://quote.alltick.co/quote-b-api/kline?token=YOUR_TOKEN&query=...
```

**WebSocket API** — 接続 URL のクエリパラメータとして渡す：

```
wss://quote.alltick.co/quote-b-ws-api?token=YOUR_TOKEN
```

## セキュリティに関する注意

- クライアントサイドのコードや公開リポジトリに Token を公開しない
