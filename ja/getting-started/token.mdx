## 申請フロー

### ステップ1 - 登録

<Info>
登録URL：[https://alltick.co/register](https://alltick.co/register)
</Info>

1. **登録ページにアクセス**：[https://alltick.co/register](https://alltick.co/register) にアクセス
2. **詳細を入力**：
   - **メールアドレス**：有効なメールアドレスを入力
   - **パスワード**：安全なパスワードを作成（大文字、小文字、特殊文字、数字を含む）
   - **認証コード**：メールに送信されたコードを入力
3. **登録完了**：「登録」ボタンをクリックし、ダッシュボードにリダイレクト

### ステップ2 - API Tokenを取得

1. **ダッシュボードに入る**：登録後、ダッシュボードに移動
2. **Tokenを見つける**：「API keys」セクションに移動
3. **Tokenを保存**：Tokenをコピーして安全に保存

## 注意事項

- Tokenを安全に保管し、信頼できない第三者と共有しない
- 登録中に問題が発生した場合は、すぐにサポートに連絡
- 常に最新のTokenを使用してAPIを呼び出す
