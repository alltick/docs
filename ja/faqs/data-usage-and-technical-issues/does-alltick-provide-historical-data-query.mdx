---
title: 'AllTick は過去データ照会を提供していますか？'
---

はい。AllTick は過去データ照会サービスを提供しています。AllTick はリアルタイム金融市場データだけでなく、外国為替、株式、商品、暗号資産など複数の資産クラスの過去データも提供します。

過去データを利用する一般的な手順は次のとおりです。

1. API ドキュメントを確認し、過去データのリクエストパラメータ、時間範囲、データ形式を理解します。
2. 有効な API キーを取得します。
3. ドキュメントに従ってリクエストを構築します。
4. リクエストを送信し、レスポンスデータを処理します。
5. データを市場トレンド分析、戦略バックテスト、市場調査などに利用します。

データのカバー範囲、リクエスト制限、データ精度に注意してください。過去データ照会で問題が発生した場合は、AllTick カスタマーサポートへお問い合わせください。
