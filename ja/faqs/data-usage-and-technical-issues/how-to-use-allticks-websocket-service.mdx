---
title: 'AllTick の WebSocket サービスを使用するには？'
---

AllTick の WebSocket サービスを使用するには、通常、次の手順を行います。

1. **WebSocket プロトコルを理解する**：WebSocket は全二重通信チャネルを提供し、クライアントとサーバー間でリアルタイムの双方向データ送信を可能にします。
2. **AllTick API ドキュメントを確認する**：WebSocket サービス関連ドキュメントを読み、接続の確立方法、リクエスト送信方法、データストリームの処理方法を理解します。
3. **API キーを取得する**：AllTick アカウントを登録し、アカウント管理または API 管理ページで API キーを取得します。
4. **WebSocket 接続コードを書く**：選択したプログラミング言語と WebSocket ライブラリを使用して接続を確立します。
5. **データリクエストを送信する**：接続確立後、ドキュメントに従って JSON 形式のデータ購読リクエストを送信します。
6. **受信データを処理する**：サーバーからプッシュされるデータをリアルタイムで受信し、画面、チャート、取引戦略などを更新します。
7. **接続を管理する**：接続終了、エラー処理、再接続ロジックを実装します。
8. **サンプルコードやライブラリを参考にする**：AllTick が提供するサンプルコードを利用して素早く開始します。

WebSocket サービスを利用する際は、リクエスト頻度制限やデータ利用ポリシーを含む AllTick の利用規約を遵守してください。
