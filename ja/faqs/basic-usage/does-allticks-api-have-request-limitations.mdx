金融市場データを提供する AllTick を含め、多くの API サービスでは、サービスの安定性と公平性を確保するために一定のリクエスト制限を設けています。制限には次のようなものがあります。

1. **レート制限（Rate Limiting）**：一定時間内に許可される最大リクエスト回数。
2. **同時実行制限**：同時に処理できるリクエスト数の上限。
3. **データ量制限**：1 回のリクエストで返されるデータポイント数、または一定期間内にダウンロードできる総データ量の制限。
4. **アクセスレベル制限**：サブスクリプションプランごとに異なるアクセス権限やリクエスト制限。

現在の制限を超える要件がある場合は、サブスクリプションプランのアップグレードを検討できます。AllTick の API リクエスト制限に関する最も正確な情報は、AllTick 公式ドキュメントまたはカスタマーサービスの説明を参照してください。
