---
title: 'データの問題や API 障害を報告するには？'
---

問題を報告する前に、可能な限り詳細な情報を収集・整理してください。たとえば次の情報が役立ちます。

* 問題が発生した具体的な日時。
* 影響を受けた API エンドポイント。
* 送信したリクエスト例。API キーなどの機密情報は削除してください。
* 受信したレスポンス。エラーコードとメッセージを含めます。
* 問題を再現する手順。

まず公式ドキュメント、FAQ、開発者コミュニティを確認し、既存の解決策がないか確認することをおすすめします。それでも報告が必要な場合は、公式サイトのお問い合わせフォーム、カスタマーサポートメール、またはサポートチケットシステムから送信し、上記の情報を提供してください。これにより技術チームが迅速に問題を特定し、解決しやすくなります。

報告時には、専門的で明確かつ完全な情報を提供することで、解決までの時間を短縮でき、製品やサービスの改善にもつながります。
