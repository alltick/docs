## 概述

AllTick REST API 文档包含 HTTP 请求示例与 HTTP接口API。HTTP接口API 页面按照官网 API 文档组织，包含 K线、批量 K线、最新盘口、最新成交价、股票产品基础信息、停复牌信息查询，以及涨跌幅、休市、假期等说明。
