AllTick 是一个金融市场数据平台，通过 REST 和 WebSocket API 提供逐笔报价、K 线历史、盘口深度和最新成交价，覆盖全球主要资产类别。

无论你在构建交易面板、运行量化策略，还是为金融科技产品提供数据支撑，AllTick 都能作为统一的数据接入层，一次集成即可获取股票、外汇、加密货币等多市场的实时和历史数据。

## 核心能力

| 能力 | 说明 |
|------|------|
| 实时行情推送 | 通过 WebSocket 流式推送，平均延迟约 170ms |
| 历史 K 线 | 支持分钟、小时、日、周、月级别 |
| 盘口深度 | 多档买卖盘数据（深度因市场而异） |
| 最新成交价 | REST 接口实时查询最新 Tick |
| 多语言示例 | Go、Python、Java、PHP 完整代码示例 |

## 接入方式

AllTick 提供两种互补的协议：

| 协议 | 适用场景 | 特点 |
|------|----------|------|
| REST API | 历史 K 线查询、一次性价格查询、服务端数据管道 | 按需拉取，简单直接 |
| WebSocket API | 实时面板、价格告警、持续数据流 | 订阅推送，无需轮询 |
