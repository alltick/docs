---
title: '认证'
description: '注册账号并获取 API Token'
---

AllTick 使用 Token 认证所有 API 请求。每个请求必须在 URL 参数中携带有效的 Token。

## 获取 Token

<Steps>
  <Step title="注册账号">
    前往 [alltick.co/register](https://alltick.co/register)，使用邮箱完成注册。
  </Step>
  <Step title="进入仪表盘">
    注册成功后自动跳转到仪表盘页面。
  </Step>
  <Step title="复制 Token">
    在仪表盘的 **API Keys** 部分找到并复制你的 Token。
  </Step>
</Steps>

## 使用方式

**REST API** — 作为 `token` 查询参数传递：

```
https://quote.alltick.co/quote-b-api/kline?token=YOUR_TOKEN&query=...
```

**WebSocket API** — 作为连接 URL 的查询参数：

```
wss://quote.alltick.co/quote-b-ws-api?token=YOUR_TOKEN
```

## 安全建议

- 不要在客户端代码或公开仓库中暴露 Token
