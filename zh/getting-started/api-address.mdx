## 股票 HTTP 接口 API 地址

基本路径: `/quote-stock-b-api`

完整 URL: `https://quote.alltick.co/quote-stock-b-api`

请求示例：

```
https://quote.alltick.co/quote-stock-b-api/kline?token=您的token&query=queryData
https://quote.alltick.co/quote-stock-b-api/batch-kline?token=您的token
https://quote.alltick.co/quote-stock-b-api/trade-tick?token=您的token&query=queryData
https://quote.alltick.co/quote-stock-b-api/depth-tick?token=您的token&query=queryData
```
