---
title: 'API 地址'
description: 'REST 和 WebSocket 端点地址一览'
---

AllTick 按资产类别划分端点。股票类使用独立的 Base URL，其他市场共用一个 Base URL。

## REST API 端点

| 资产类别 | Base URL |
|----------|----------|
| 美股、港股、A 股、指数 | `https://quote.alltick.co/quote-stock-b-api` |
| 外汇、加密货币、贵金属、商品 | `https://quote.alltick.co/quote-b-api` |

可用路径：

| 路径 | 方法 | 说明 |
|------|------|------|
| `/kline` | GET | 单产品历史 K 线查询 |
| `/batch-kline` | POST | 批量查询最新 2 根 K 线 |
| `/trade-tick` | GET | 最新成交价查询 |
| `/depth-tick` | GET | 最新盘口深度查询 |
| `/static_info` | GET | 股票产品基础信息批量查询，仅股票类支持 |
| `/api/suspension/sse` | GET | 上海证券交易所停复牌信息查询 |
| `/api/suspension/nyse` | GET | 纽约证券交易所停复牌信息查询 |
| `/api/suspension/nasdaq` | GET | 纳斯达克交易所停复牌信息查询 |

请求示例：

```
https://quote.alltick.co/quote-stock-b-api/kline?token=YOUR_TOKEN&query=URL_ENCODED_JSON
```

## WebSocket 端点

| 资产类别 | 连接地址 |
|----------|----------|
| 美股、港股、A 股、指数 | `wss://quote.alltick.co/quote-stock-b-ws-api?token=YOUR_TOKEN` |
| 外汇、加密货币、贵金属、商品 | `wss://quote.alltick.co/quote-b-ws-api?token=YOUR_TOKEN` |

<Note>
查询外汇或加密货币品种时，使用 `/quote-b-api`（REST）或 `/quote-b-ws-api`（WebSocket）。查询股票时使用带 `-stock-` 的路径。请求参数结构完全相同，只需切换 Base URL。
</Note>

<Note>
`/static_info` 和 `/api/suspension/*` 为股票相关接口。`/static_info` 使用 `https://quote.alltick.co/quote-stock-b-api/static_info`，停复牌接口使用 `https://quote.alltick.co/api/suspension/{sse|nyse|nasdaq}`。
</Note>
