---
title: '心跳'
description: '保持 WebSocket 连接的心跳机制'
---

每 10 秒发送一次心跳消息。30 秒内未收到心跳，服务端将断开连接。

## 请求（cmd_id: 22000）

```json
{
  "cmd_id": 22000,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```

## 响应（cmd_id: 22001）

```json
{
  "ret": 200,
  "msg": "ok",
  "cmd_id": 22001,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```
