---
title: '取消订阅'
description: '取消已订阅的行情推送'
---

取消已订阅的报价推送。

## 请求（cmd_id: 22006）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| cmd_id | integer | 是 | 固定值 22006 |
| seq_id | integer | 是 | 序列号 |
| trace | string | 是 | 追溯 ID |
| data.cancel_type | integer | 是 | 0=全部，1=盘口，2=成交，3=汇率 |

```json
{
  "cmd_id": 22006,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {
    "cancel_type": 1
  }
}
```

## 响应（cmd_id: 22007）

```json
{
  "ret": 200,
  "msg": "ok",
  "cmd_id": 22007,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```
