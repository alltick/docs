## 接口说明

取消报价订阅

## 请求-协议号：22006

#### data定义

| 字段 | 名称 | 类型 | 必填项 | 说明 |
| --- | --- | --- | --- | --- |
| cancel\_type | 取消类型 | uint32 | 是 | 0：取消所有报价订阅，1：取消盘口报价订阅，2：取消成交报价订阅，3：取消汇率订阅 |

### 数据结构(json)

```json
{
    "cmd_id":22006,
    "seq_id":123,
    "trace":"3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
    "data":{
        "cancel_type": 1
    }
}
```
