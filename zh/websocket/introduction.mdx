---
title: 'WebSocket API 概述'
description: '实时行情数据推送接口概述'
---

AllTick WebSocket API 提供实时行情数据推送服务。连接后订阅感兴趣的品种，即可持续接收最新成交价和盘口深度数据。

## 协议号一览

| 协议号 | 说明 |
|--------|------|
| 22000/22001 | 心跳（Ping/Pong） |
| 22002/22003 | 盘口订阅 |
| 22004/22005 | 成交价订阅 |
| 22006/22007 | 取消订阅 |
| 22998 | 成交价推送 |
| 22999 | 盘口推送 |

## 接口地址

| 资产类别 | 连接地址 |
|----------|----------|
| 美股、港股、A 股、指数 | `wss://quote.alltick.co/quote-stock-b-ws-api?token=YOUR_TOKEN` |
| 外汇、加密货币、贵金属、商品 | `wss://quote.alltick.co/quote-b-ws-api?token=YOUR_TOKEN` |

## 使用说明

1. 建立连接时在 URL 中附加 Token 参数
2. 连接成功后，每 **10 秒**发送一次心跳
3. **30 秒**内未收到心跳，服务端将断开连接
4. 每次订阅请求会覆盖上一次（追加品种需重新发送完整列表）
5. 客户端应实现自动重连逻辑
