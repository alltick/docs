## 概述

AllTick WebSocket API 提供实时行情数据推送服务：

| 协议号 | 说明 |
|--------|------|
| 22000/22001 | 心跳 |
| 22002/22003 | 最新盘口订阅 |
| 22004/22005 | 最新成交价订阅 |
| 22006/22007 | 取消报价订阅 |
| 22998 | 成交价推送 |
| 22999 | 盘口推送 |

## 接口地址

**股票类（美股、港股、A股）：**

```
wss://quote.alltick.co/quote-stock-b-ws-api?token=您的token
```

**外汇、加密货币、贵金属、商品：**

```
wss://quote.alltick.co/quote-b-ws-api?token=您的token
```

## 使用说明

1. 建立 WebSocket 连接时，必须在 URL 中附加认证 token
2. 连接成功后，每 **10 秒**发送一次心跳
3. **30 秒**内未收到心跳将断开连接
4. 每次发送订阅请求会覆盖上一次订阅（追加订阅需重新发送完整列表）
5. 建议客户端增加断开自动重连逻辑
