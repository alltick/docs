---
title: '盘口订阅'
description: '订阅实时盘口深度数据'
---

订阅产品的最新盘口（Order Book）数据。每次订阅请求会覆盖上一次。

各市场最大盘口深度：

| 产品类型 | 最大深度 |
|----------|----------|
| 外汇、贵金属、原油 | 1 档 |
| 加密货币 | 5 档 |
| 港股 | 10 档 |
| 美股 | 1 档 |
| 沪深 A 股 | 5 档 |

## 请求（cmd_id: 22002）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| cmd_id | integer | 是 | 固定值 22002 |
| seq_id | integer | 是 | 序列号 |
| trace | string | 是 | 追溯 ID |
| data.symbol_list | array | 是 | 产品列表 |
| data.symbol_list[].code | string | 是 | 产品代码 |
| data.symbol_list[].depth_level | integer | 否 | 深度层级（默认 1） |

```json
{
  "cmd_id": 22002,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {
    "symbol_list": [
      { "code": "BTCUSDT", "depth_level": 5 },
      { "code": "ETHUSDT", "depth_level": 5 }
    ]
  }
}
```

## 响应（cmd_id: 22003）

```json
{
  "ret": 200,
  "msg": "ok",
  "cmd_id": 22003,
  "seq_id": 123,
  "trace": "3baaa938-f92c-4a74-a228-fd49d5e2f8bc-1678419657806",
  "data": {}
}
```

## 推送（cmd_id: 22999）

| 字段 | 类型 | 说明 |
|------|------|------|
| code | string | 产品代码 |
| seq | string | 报价序号 |
| tick_time | string | 时间戳（毫秒） |
| bids | array | 买盘 [{price, volume}] |
| asks | array | 卖盘 [{price, volume}] |

```json
{
  "cmd_id": 22999,
  "data": {
    "code": "BTCUSDT",
    "seq": "1605509068000001",
    "tick_time": "1605509068",
    "bids": [
      { "price": "9.12", "volume": "100" }
    ],
    "asks": [
      { "price": "9.13", "volume": "150" }
    ]
  }
}
```
